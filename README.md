## 🌟✨ 项目介绍 ✨🌟

`Qazr Addons` 是面向 **Minecraft 1.21.11** 与 **Meteor Client** 的客户端扩展源码集合。仓库包含战斗、移动、聊天、采矿、创造物品以及自定义药水等模块源码。⚡🧩🔥

> 📌 当前仓库直接保存 `main/java` 与 `main/resources` 源码树。

---

## 🧩🎮 功能模块 🎮🧩

| 图标 | 模块 | 功能简介 |
| :---: | --- | --- |
| ⚔️ | **MaceKill** | 重锤战斗、目标筛选、移动与攻击流程 |
| 💥 | **MaceDMG** | 重锤伤害与移动数据包逻辑 |
| ⛏️ | **AutoMine** | 矿物扫描、路线规划、移动和自动挖掘 |
| 💬 | **AutoGG** | 战斗消息识别与自动 GG |
| 📣 | **AutoFuck** | 自动聊天、目标选择与文本处理 |
| 🎁 | **CreativeGive** | 创造模式物品、属性与附魔处理 |
| 🧪 | **CustomPotion** | 自定义药水类型、效果、颜色与持续时间 |
| 🎯 | **Targeting** | 玩家与实体目标筛选 |
| 🌀 | **Movement** | 坐标移动、旋转和数据包处理 |
| 🎒 | **Inventory** | 物品栏、槽位与重锤切换逻辑 |

## ⚠️📚 使用提示 📚⚠️

- 🔧 当前仓库以源码和资源文件为主。
- 📦 构建时需要自行配置 Fabric Loom、Minecraft 1.21.11、Yarn mappings 和 Meteor Client 依赖。
- 🧪 建议先在本地测试环境检查兼容性与运行效果。
- 🛡️ 使用数据包、移动和战斗功能时，请留意目标服务器规则。
- 💾 修改前建议保留备份，并使用 Git 记录每次变更。
